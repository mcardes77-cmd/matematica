# Math Arena Insana 3.0 Completa

Pacote completo com:
- 16 temas
- 100 perguntas por tema
- níveis fácil, médio, difícil e misto
- ordem sem repetição até acabar o baralho
- modo campeonato
- passar (-2) e roubo com erro (-5)
- sons e animações
- editor de perguntas
- geração com IA
- histórico local
- exportação para Excel

## Como rodar
pip install flask openpyxl openai
python app.py

## Acessos
http://127.0.0.1:5000
http://127.0.0.1:5000/editor

## Arquivos importantes
- Banco local: matharena.db
- Perguntas: perguntas.json
